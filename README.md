## ğŸ¨ Customization

### Colors
Edit the CSS variables in `assets/css/style.css`:

```css
:root {
    --primary-color: #d4a574; /* Rose Gold */
    --secondary-color: #6b4c7a; /* Deep Plum */
    --accent-color: #f8f4f1; /* Ivory */
    --text-dark: #2c2c2c; /* Charcoal */
    /* ... more variables */
}
```

### Typography
Font families are defined at the top of the CSS file. You can change them by updating the Google Fonts import and the font-family properties.

### Layout
All pages use a consistent 6-section structure. Modify the section classes to change backgrounds and layouts.
